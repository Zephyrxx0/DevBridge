# Phase 28 — UI Design Contract

> Visual and interaction contract for the UI overhaul including landing page and AI-native chat interface. Generated by gsd-ui-researcher.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |
| Preset | base-nova |
| Component library | Radix UI + Base UI |
| Icon library | Lucide |
| Font | Geist Sans (Primary), Geist Mono (Code) |

### Visual Hierarchy
- **Primary Focal Point:** The focused message stream in the center of the chat interface.
- **Secondary Focal Point:** The "Start Building" call-to-action in the hero section.
- **Tertiary Focal Point:** The thread history sidebar for contextual navigation.

---

## Spacing Scale

Declared values (strictly following 8-point grid principle, aligned with standard set):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight list items |
| sm | 8px | Button padding, component internal spacing |
| md | 16px | Card padding, default gap between elements |
| lg | 24px | Layout gaps, sidebar padding, section margins |
| xl | 32px | Container padding, major component spacing |
| 2xl | 48px | Landing page section spacing |
| 3xl | 64px | Page-level vertical margins |

*No exceptions allowed for this phase. All spacing must snap to these tokens.*

---

## Typography

Strictly limited to 4 sizes and 2 weights (even-numbered for grid alignment).

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Label | 14px | 400 | 1.4 |
| Body | 16px | 400 | 1.5 |
| Heading | 24px | 600 | 1.2 |
| Display | 48px | 600 | 1.1 |

Note: `text-hero` uses `clamp(3rem, 6vw, 4rem)` for responsive display, snapping to 48px base.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `oklch(0.08 0.005 45)` | Near-Black Background / Main Surface |
| Secondary (30%) | `oklch(0.11 0.007 45)` | Cards, Sidebar, Popovers (Surface-1/2) |
| Accent (10%) | `oklch(0.62 0.2 38)` | Orange Flame / Primary Buttons / Glows |
| Destructive | `oklch(0.6 0.18 20)` | Accent Rose / Error States / Delete Actions |

Accent reserved for: Primary action buttons, active sidebar states, brand marks, dithered card highlights, and focused chat input borders.
