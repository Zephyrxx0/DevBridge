## Visual Hierarchy

The primary screen for the Admin Dashboard focuses on presenting repository-specific AI-generated "intern confusion" reports.
- **Focal Point:** The Markdown Feed list containing the pre-computed reports.
- **Secondary Elements:** Repository filter controls and the top-level page heading.
- **Layout:** A single-column or wide content layout to prioritize readability of the markdown reports.
