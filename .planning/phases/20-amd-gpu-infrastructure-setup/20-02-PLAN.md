  <name>Task 1: Docker Volume Cache Binding</name>
  <files>docker-compose.yml</files>
  <action>
    Update `docker-compose.yml` to persist `/root/.cache/huggingface` in host volumes, mapping the host's UID/GID to prevent root privilege issues (per D-11).
    Use isolated directories for the models to prevent lock contention (per D-09). Do not add cron scripts for cache eviction (per D-10).
    For `vllm-deep`: Add `volumes: ["/app/repo_cache/qwen:/root/.cache/huggingface"]` and `user: "${UID}:${GID}"`.
    For `vllm-fast`: Add `volumes: ["/app/repo_cache/gemma:/root/.cache/huggingface"]` and `user: "${UID}:${GID}"`.
  </action>
  <verify>

  </verify>
